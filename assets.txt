Usage Guide:
DISABLEING PARENTAL CONTROL
--parental_control_disabler.cmd /do

ENABLING PARENTAL CONTROL
--parental_control_disabler.cmd /undo
This command will reverse the action previously performed.

ENABLE DEBUG(FOR LOGGING)	
--parental_control_disabler.cmd /do /on
--parental_control_disabler.cmd /undo /on
Adding the /on argument will turn on debug mode, which helps you see the detailed process of your script as it runs.

HELP :
--parental_control_disabler.cmd
--parental_control_disabler.cmd /showreadme

If you don't enter any arguments, the script will display a help message (i.e., the printhelp command in your code).
